# kaggle-challenges
A collection of Machine Learning solutions for Kaggle competitions. Focused on building robust scikit-learn pipelines, with an upcoming focus on behavioral and criminological data analysis.

## 🚀 Future Roadmap

- [x] **Phase 1 (Current):** Master fundamental ML workflows (Titanic, House Prices) using **scikit-learn Pipelines** and Ensembles (RF/XGBoost).
- [ ] **Phase 2 (Upcoming):** Apply ML to **Behavioral Science**. Analyzing datasets related to human decision-making and social patterns.
- [ ] **Phase 3 (Long-term Goal):** Bridging **Criminal Psychology** and Data Science to develop predictive models for public safety and platform integrity.
